# 표(Table)

- position 속성  
- 콜론 기호를 통해서 가운데/왼쪽/오른쪽 정렬 가능

값 | 의미 | 기본값
--|:--:|--:
static | 기준 없음 | O
relative | 요소 자신 | X
absolute | 위치 상 부모 요소 | X
fixed | 뷰포트 | X
